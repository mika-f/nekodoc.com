# Project: NekoDoc

NekoDoc is a static site generator that focused on project documentation, api documentation, and product documentation.  
Documentation sources are written in Markdown and MDX (React), and zero-configuration start-up.

NekoDoc is open-source software hosted on [GitHub](https://github.com/mika-f/nekodoc), licensed under the MIT License.

## Features

- zero-configuration startup by `nekodoc init`
- fully customizable
- TailwindCSS included
- Markdown (`.md`) and MDX (React) (`*.mdx`) support
- TypeScript (`*.tsx`) support
